### Flash Backup Beta ###

**A utility to backup the unRAID flash drive**

## Features ##
- Backing up to local storage
- Backing up to cloud storage using rclone (can use the plugin, any of the current docker containers, or the built in rclone if using unraid 7.3+)
- Confirmed working with mega, box, onedrive, google drive, backblaze b2, dropbox, and s3 providers cloud storages
- Offers scheduling of both local and remote cloud storage backups
- Ability to set how many backups you want to keep
- Dry run support
- Notification support for discord, gotify, ntfy, pushover, slack, and unraids built in
- Ability to set an owner for your local backups
- Logging available in the webui and at /tmp/flash-backup_beta
